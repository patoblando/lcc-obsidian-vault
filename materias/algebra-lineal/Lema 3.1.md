# Lema

Sea $V$ e.v sobre $\mathbb{K}$, $S, T_{1}, T_{2} \in \mathcal{L}(V)$, $\alpha \in \mathbb{K}$ luego valen,

1. $IS = SI = S$
2. $S(T_{1} + T_{2}) = ST_{1} = ST_{2}$ ; $(T_{1} + T_{2})S = T_{1}S + T_{2}S$
3. $\alpha(ST_{1}) = (\alpha S)T_{1} = S (\alpha T_{1})$
